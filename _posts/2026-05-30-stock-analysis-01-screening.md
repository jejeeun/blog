상장사 2,000개를 전부 분석하기는 어렵다. 그래서 숫자 몇 개로 먼저 후보를 추린다. 이 과정을 스크리닝이라고 한다.

스크리닝을 통과한 종목은 매수 대상이 아니라 더 살펴볼 후보다. 완벽한 회사를 고르는 단계가 아니라, 들여다볼 가치가 있는 회사를 추리는 단계다.

## 무엇을 보나

### 시가총액

주가에 총 주식 수를 곱한 값으로, 회사 전체의 몸값이다. 시총이 너무 작은 회사(예: 1,000억 미만)는 거래가 적어 가격이 크게 흔들린다. 어느 정도 규모가 있는 회사부터 보는 편이 안전하다.

### PER

주가를 1주당 1년 순이익으로 나눈 값이다. 몇 년이면 투자금을 회수하는지를 뜻하며, 낮을수록 싸다.

![PER 개념 — 본전 뽑는 햇수](per-concept.svg){: w="680" }

1만 원짜리 회사가 1년에 1,000원을 벌면 10년이면 본전이고, PER은 10이다. 옆 회사가 500원밖에 못 벌어 20년이 걸리면 PER은 20으로, 같은 값에 더 비싸게 산 셈이다. 업종마다 정상 범위가 달라서 같은 업종끼리 비교한다.

### PBR

주가가 회사 순재산(자산에서 빚을 뺀 값)의 몇 배인지를 나타낸다.

```
순재산이 1주당 10,000원일 때
 주가 8,000원  → PBR 0.8  (재산보다 싸다)
 주가 10,000원 → PBR 1.0  (재산만큼)
 주가 20,000원 → PBR 2.0  (재산의 2배 값)
```

PBR이 1보다 작으면 회사 재산보다 싸게 팔리고 있다는 뜻이다.

### ROE

1년 순이익을 자기자본으로 나눈 값으로, 자기 돈으로 얼마나 잘 버는지를 보여 준다.

```
1억을 넣어 1년에
 2,000만 원 → ROE 20%  (잘하는 편)
 1,000만 원 → ROE 10%  (괜찮은 편)
   300만 원 → ROE 3%   (은행 이자 수준)
```

ROE가 높고 꾸준하면 자본을 잘 굴리는 회사다.

### 부채비율

빚을 자기자본으로 나눈 값이다. 낮을수록 안전하며, 100% 안팎 이하가 무난한 기준이다.

## 거르는 기준

가치주 후보를 찾는 기본 조합은 다음과 같다.

```
✅ PBR 1.0 미만       → 싸다
✅ ROE 10% 이상       → 잘 번다
✅ 부채비율 100% 미만  → 안전하다
✅ 시총 1,000억 이상   → 너무 작지 않다
```

네 조건을 모두 통과하면 더 살펴볼 후보가 된다.

> 숫자가 좋다고 반드시 좋은 회사는 아니다. 싼 데는 이유가 있을 수 있다(밸류 트랩). 이 판단은 사업과 재무를 직접 들여다본 뒤에 가능하다.

## 어디서 보나

네이버 금융(finance.naver.com)에서 종목을 검색하면 PER·PBR·ROE·시가총액이 한 화면에 나온다. 여러 종목을 같은 기준으로 비교하며 메모하면 된다.

종목을 한꺼번에 거를 때는 KRX 정보데이터시스템에서 표로 내려받거나, pykrx로 코드 한 줄에 전 종목의 숫자를 받아 자동으로 거를 수 있다.

## 결과물

스크리닝의 산출물은 거창한 보고서가 아니라 한 줄짜리 후보 목록이다.

```
- A전자 : PBR 0.7 / ROE 12% / 부채 60%  → 1순위
- B화학 : PBR 0.9 / ROE 11% / 부채 80%  → 2순위
- C건설 : PBR 0.5 인데 ROE 4%           → 보류 (왜 싼지 확인 필요)
```

이렇게 추린 후보 중 위 순위대로 한 종목씩 사업 구조부터 깊이 들여다본다.
